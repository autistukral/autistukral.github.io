1. Download WinRAR and Install
Or just go here: https://www.rarlab.com/download.htm
download the newest version under the "English WinRAR and RAR release" section
2. In File Epxlorer on Windows 11 23H2 and higher press "Ctrl + T"
To open a new tab the same way you open a new tab in web browsers
3. Go to: "C:\Program Files\WinRAR"
You can copy the adress and paste it into the File Epxlorer adress bar (not the search)
4. Copy the "rarreg.key" from Fuck off Windows / WinRar
5. Put the copied "rarreg.key" into the WinRAR folder
